# It_project (Blind Sensor Band)

CODE and Important images, presentation, etc for the project.

>Important pins

- UltraSonic Sensor
	- Trigger (10) 
	- Echo (13)
	- GND (GND)
	- VCC (5v)

- Buzzer
	- Positive (4)
	- Negative (GND)

- Vibrator
	- Positive (12)
	- Negative (GND)

- GPS chip
	- TX (2)
	- RX (3)
	- VCC (5v)
	- GND (GND)

- GSM 800l
	- TX (8)
	- RX (9)
	- VCC (out + of bug convertor)
	- GND (GND)

- Bug Convertor
	- Out- (GND)
	- Out+ (Vcc of GSM)
	- In- (Battery-)
	- In+ (Battery+)
